Project Overview

Created a tool that predicts the images of certain sports celebrities

Notes

  • Data Cleaning
  • Model Building
  • Backend
  • Resources Used

Cleaning the Data

Here the images has been cropped and resized and sorted in respective folders.



Model Building

Here the data is split into train and tests sets with a test size of 30%. We tried four different models and tuned them using GridSearchCV. We further evaluated them using Accuracy matrics.

  • svm : 0.870968
  • logistic_regression : 0.851613
  • random_forest : 0.825806
  • k_neighbor : 0.767742
  • decision_tree : 0.574194



Backend

The web server is built on flask API endpoint that was hosted on a local webserver. The API endpoint takes in a request as image and returns a prediction. Finally we gave it a form of a web application.



Codes and Resources Used

  • Python Version: 3.8.5
  • Scikit-Learn Version : 2.2.0
  • Packages: numpy, matplotlib, flask, pandas, Opencv, scikit-learn
  • For Web Framework Requirements: pip install -r requirements.txt
  • This Dataset is scrapped from google images using Selenium.
  • For UI Design: Figma
